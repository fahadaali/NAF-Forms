# نظام استبانات ناف — NAF Forms

منصة موحّدة لبناء **الاختبارات**، و**التقديم الوظيفي (رفع السيرة الذاتية)**،
و**الاستبيانات والاستطلاعات** — مع أنواع بيانات متعددة، وقوالب جاهزة، ولوحة
ردود تفصيلية، وتصدير بأكثر من صيغة.

## المميزات

- **أنواع نماذج**: استبيان/استطلاع، اختبار (مع تصحيح تلقائي)، تقديم وظيفي.
- **١٦ نوع بيانات (حقول)**: نص قصير، فقرة، رقم جوال، بريد إلكتروني، رقم،
  تاريخ، اختيار من متعدد، مربعات اختيار، قائمة منسدلة، مقياس خطي، تقييم شبكي،
  تقييم بالنجوم (٥ نجوم)، موقع على الخريطة، عنوان سكن، رفع ملف، عناوين/فواصل.
- **الإلزامي والاختياري** مع تمييز الحقل الإلزامي بنجمة.
- **بنّاء نموذج** كامل مع إعداد مستقل لكل حقل، و**قوالب جاهزة** للاستخدام السريع.
- **تخصيص صفحة التقديم**: العنوان، المحتوى، الألوان، الصور، الشعار، مقاطع
  يوتيوب داخل الصفحة، الروابط، والملفات (مع التحكم بإمكانية التنزيل)، ورسالة
  ما بعد الإرسال.
- **صفحة تقديم على شكل بطاقات**: سؤال لكل بطاقة، انتقال سلس للتالي، وإمكانية
  الرجوع للسؤال السابق، وشريط تقدّم — بواجهة عربية RTL.
- **المشاريع**: لكل مشروع عدة نماذج/استبيانات بإعدادات مستقلة.
- **لوحة ردود تفصيلية** متكيّفة حسب نوع البيانات (رسوم بيانية للاختيارات
  والمقاييس، متوسطات، عرض المواقع والملفات)، مع تسجيل **تاريخ ووقت** كل رد.
- **تصدير الردود** بصيغتي CSV و JSON.
- **منطق شرطي**: إظهار/إخفاء الأسئلة بناءً على إجابات سابقة.
- **التحكم بالاستقبال**: حد أقصى لعدد الردود وتاريخ إغلاق تلقائي.
- **حماية بكلمة مرور** للنموذج، و**جمع بريد المستفيد**.
- **صلاحيات المشرفين**: تسجيل دخول يحمي لوحة التحكم (صفحة التقديم تبقى عامة).
- **إشعار بريد** للمشرف عند وصول رد جديد (عبر SMTP).

## التقنيات

Next.js 14 (App Router) · TypeScript · Tailwind CSS · Prisma · SQLite · Leaflet

## التشغيل محليًا

```bash
npm install                # يُنشئ Prisma Client تلقائيًا
echo 'DATABASE_URL="file:./dev.db"' > .env
npm run db:push            # إنشاء قاعدة البيانات
npm run db:seed            # تحميل القوالب الجاهزة
npm run dev                # التطوير على http://localhost:3000
```

للإنتاج:

```bash
npm run build && npm run start
```

## إعدادات البيئة (.env)

| المتغير | الوصف | الافتراضي |
| --- | --- | --- |
| `DATABASE_URL` | مسار قاعدة SQLite | `file:./dev.db` |
| `ADMIN_PASSWORD` | كلمة مرور دخول لوحة التحكم | `naf-admin` |
| `SMTP_HOST` | خادم البريد (لإشعارات الردود) | — |
| `SMTP_PORT` | منفذ SMTP (465 للتشفير) | — |
| `SMTP_USER` / `SMTP_PASS` | بيانات اعتماد SMTP | — |
| `SMTP_FROM` | عنوان المرسِل | `SMTP_USER` |

> صفحة التقديم `/f/[slug]` عامة للجميع، أما بقية الصفحات فمحمية بتسجيل دخول
> المشرف. إشعارات البريد تُرسل فقط عند ضبط متغيرات `SMTP_*`، وإلا تُتجاهل بهدوء.

## بنية المشروع

```
prisma/schema.prisma        نموذج البيانات: Project · Form · Question · Response · Answer
prisma/seed.ts              القوالب الجاهزة (اختبار، تقديم وظيفي، رضا الموظفين، استطلاع)
src/lib/field-types.ts      سجل أنواع الأسئلة (الحقول)
src/lib/types.ts            إعدادات التخصيص (FormSettings) و DTOs
src/components/             مكوّنات: بنّاء النموذج، صفحة التقديم، لوحة الردود، إدخال الحقول
src/app/                    الصفحات ومسارات الـ API
  api/                      projects · forms · upload · submit · export
  f/[slug]/                 صفحة التقديم العامة (بطاقات)
  forms/[formId]/edit/      بنّاء النموذج والتخصيص
  forms/[formId]/responses/ لوحة الردود والتحليلات
  projects/[projectId]/     نماذج المشروع
```

## المسارات الرئيسية

| المسار | الوصف |
| --- | --- |
| `/` | الرئيسية: المشاريع والقوالب الجاهزة |
| `/projects/[id]` | نماذج المشروع |
| `/forms/[id]/edit` | بنّاء النموذج والتخصيص والنشر |
| `/forms/[id]/responses` | لوحة الردود والتحليلات والتصدير |
| `/f/[slug]` | صفحة التقديم العامة (رابط المشاركة) |
